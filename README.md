# Vrun

Vrun (Virtual [Code] Runner) is an online development environment and code runner, mainly built for evaluation, contests, 
and code interview.

Scripting Pipeline:
- Pre-script -> preparation sblm code jalan.
- Code execution.
- Post-script -> cek/validasi output setelah code jalan -> bisa jadi output utk testcase.
- Validation script -> cek testcase custom
